# Asset Placeholders

## chimera.png
Place the background image `chimera.png` in this directory.
The CSS references it at `/assets/images/chimera.png`

Recommended specifications:
- Resolution: 1920x1080 or higher
- Format: PNG or WebP
- Theme: Dark, futuristic, horse racing / AI themed
- Colors: Deep blues, purples, cyan accents matching the CSS palette

## chimera1-bg.mp4
Place the intro video in `/assets/video/chimera1-bg.mp4`

Recommended specifications:
- Duration: 5-10 seconds
- Resolution: 1920x1080
- Format: MP4 (H.264)
- Theme: Dramatic intro animation matching the brand
