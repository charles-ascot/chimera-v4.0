# Chimera v4.0 Frontend

React/Vite UI using the **exact** `src/styles/main.css` supplied by Charles.

## Background image
Drop your image here:

- `public/static/assets/images/chimera.png`

## Intro video
Drop your intro video here:

- `public/static/assets/videos/chimera1-bg.mp4`

Notes:
- The intro plays once per browser (tracked via `localStorage: chimera_intro_seen`).
- Click anywhere on the intro to skip.

## Local dev

```bash
npm install
VITE_API_BASE_URL=http://localhost:8080 npm run dev
```
